LA High Tech Bridge & Coding Boot Camp

The Bridge to Coding training is comprised of a two-week session where students are prepared to enter the Coding Boot Camp. Students will be exposed to important aspects of being a new employee that includes: Workplace Success, Shining as a New Employee, Oral and Written Communication for the IT office environment, Workplace Behavior, Emotional Intelligence, Cultural Diversity, and Interviewing skills. The 16-Week Coding Boot Camp is designed to prepare 135 programmers and codes to transition into job openings in the Information and Communication Technologies (ICT) industry. Training in the LA High Tech Bridge & Coding Boot Camp includes a 1-day paid internship!

Eligibility Requirements

  • 18 years or older
  • Completed 1 level below college English
  • Completed college level Math
  • Completed basic computer literacy
Each area must be completed with a Grade "C" or better or assessed @ 70% or higher. Verification of Eligibility Completions are required for registration and enrollment!

BootCamp Skills

Students completing the Coding BootCamp will be able to:

  • Develop web applications using a web based full stack programming environment
  • Design, implement and test web applications
  • Design web user interfaces using HTML, CSS, and JavaScript
  • Use web services to transfer data and add interactive components to web pages
